# 🌊 파주시 긴급 대피 경로 시스템 (SafeRoute)

## 🚀 배포 주소 (Live Demo)
👉 **[서비스 바로가기 (https://saferoute-delta.vercel.app/)](https://saferoute-delta.vercel.app/)**
> 별도의 설치 없이 위 링크를 클릭하면 즉시 시뮬레이션을 체험하실 수 있습니다.

---

## 1. 프로젝트 개요
**"지금 100mm의 폭우가 쏟아진다면, 당신은 어디로 대피하시겠습니까?"**

이 프로젝트는 파주시의 **실제 과거 침수 흔적 데이터**와 **실시간 기상 데이터**를 결합하여, 극한 호우 발생 시 안전한 대피 경로를 안내하는 재난 시뮬레이션 웹 서비스입니다. 단순한 대피소 위치 안내를 넘어, 강수량에 따라 확장되는 위험 구역을 시각화하고 이를 회피하는 **최적의 생존 경로**를 제시합니다.

## 2. 문제 정의 (Problem Definition)
* **기후 위기의 현실화**: 2024년, 2025년 연이은 집중 호우로 파주시(공릉천, 문산천 일대)의 침수 피해가 반복되고 있습니다.
* **정보의 한계**: 기존 재난 문자는 "대피하라"는 경고만 줄 뿐, **"침수된 도로를 피해 어디로 가야 하는지"** 에 대한 구체적인 솔루션은 부재합니다.
* **우리의 해결책**: 사용자가 직접 강수량을 입력해 미래 상황을 예측해보고(Vibe Coding), 침수되지 않은 **'진짜 안전한 길'** 을 찾을 수 있도록 돕습니다.

## 3. 주요 기능 (Key Features)
1.  **🌤️ 실시간 기상 데이터 연동**: 별도 조작 없이 접속과 동시에 파주시의 현재 날씨와 강수량을 로드하여 상황을 알려줍니다.
2.  **🌊 사용자 참여형 시뮬레이션**:
    * 사용자가 강수량(mm/h)을 입력하면 **Turf.js** 알고리즘이 작동하여 위험 구역을 동적으로 확장합니다.
    * **30mm 미만**: 안전 (Safe)
    * **30~80mm (주의보)**: 과거 침수 이력 지역 위주로 경고 표시
    * **80mm 이상 (극한호우)**: 주변 도로 및 저지대까지 침수 구역 대폭 확장 및 병합 (Union)
3.  **🛡️ 스마트 대피소 필터링**:
    * 위험 구역 내부에 위치한 대피소는 즉시 **'폐쇄(Closed)'** 처리하고 회색으로 비활성화합니다.
    * 사용자 위치에서 가장 가깝고 **안전한(Operating)** 대피소 3곳만 선별하여 추천합니다.
4.  **📍 생존 경로 탐색**: OSRM API를 활용하여, 물에 잠긴 위험 지역을 우회하는 최단 도보/차량 경로를 지도에 그립니다.

## 4. 활용 데이터 (Data Sources)
이 프로젝트는 신뢰할 수 있는 공공데이터를 기반으로 분석되었습니다.
* **침수흔적도 (`spggcee:tm_fldn_trce`)**: 경기데이터드림 (파주시 과거 실제 침수 구역 Shape 데이터)
* **이재민 임시거주시설 (`spggcee:dsvctm_tmpr_hab_fclt`)**: 경기데이터드림 (대피소 위치 정보)
* **기상 정보**: Open-Meteo Free Weather API (실시간 강수량 및 날씨 코드)
* **지도 및 지리정보**: OpenStreetMap, 파주시 행정경계 GeoJSON

## 5. 기술 스택 (Tech Stack)
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
* **Map Engine**: Leaflet.js
* **Spatial Analysis**: Turf.js (폴리곤 버퍼링, 유니온 등 공간 연산), Proj4js (좌표계 변환 EPSG:5179 → WGS84)
* **Deployment**: Vercel (Static Web Hosting)

## 6. 실행 방법
**[권장] 위 배포 주소(Live Demo)를 통해 접속하는 것을 권장합니다.**

로컬 환경에서 실행하려면:
1.  이 저장소를 다운로드하거나 클론합니다.
2.  `index.html` 파일을 실행합니다.
    > ⚠️ **주의**: 브라우저 보안 정책(CORS) 및 GeoJSON 로드를 위해, VS Code의 **Live Server** 확장 프로그램을 사용하거나 로컬 웹 서버 환경에서 실행해야 정상 작동합니다.
3.  지도의 빈 곳을 클릭하여 **내 위치**를 설정합니다.
4.  우측 입력창에 **강수량(예: 120)** 을 입력하고 **[시뮬레이션 실행]** 버튼을 누릅니다.
5.  붉게 확장되는 침수 구역을 피해 생성되는 안전 경로를 확인합니다.

## 7. 프로젝트의 차별점
* **살아있는 데이터**: 더미 데이터가 아닌, API를 통해 받아온 **5,000건 이상의 실제 데이터**를 실시간으로 처리합니다.
* **시각적 스토리텔링**: 위험 구역에 맥박이 뛰는 듯한 애니메이션(Pulse Effect)을 넣어 재난의 긴급함을 시각적으로 전달했습니다.
* **기술적 도전**: 서로 다른 좌표계(5179, 4326)의 데이터를 통합하고, 수천 개의 폴리곤을 실시간으로 병합(Union)하는 고급 공간 분석 기술을 적용했습니다.
